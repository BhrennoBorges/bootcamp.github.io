Meus Projetos

Aqui estão alguns dos projetos que desenvolvi, mostrando minha evolução nas áreas de programação e automação.

Desenvolvimento de Bot para Discord

Desenvolvi um bot personalizado para Discord, utilizando programação em Python e integração com a API do Discord. O bot foi projetado para automatizar tarefas, moderar o servidor e melhorar a interação entre os usuários, incorporando funcionalidades como comandos personalizados, respostas automatizadas e gestão de permissões. O projeto me permitiu aprimorar minhas habilidades em programação e automação de processos, além de integrar soluções criativas em um ambiente de chat.

Desenvolvimento de To-Do List em Python

Criei um simples aplicativo de To-Do List utilizando Python, focado na organização de tarefas diárias. O sistema permite adicionar, visualizar e remover tarefas através de um menu interativo. O projeto envolveu a utilização de estruturas de dados como listas e condicionais (if, elif, else), além de loops para garantir a navegação contínua no aplicativo. Este projeto aprimorou minhas habilidades em lógica de programação e manipulação de entradas do usuário.
